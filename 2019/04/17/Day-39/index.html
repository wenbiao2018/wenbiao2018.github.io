总结

  1. 声明数组时不能指定其长度，因为数组是一种引用型变量，所以声明时仅仅表示定义了一个引用变量，还未指向任何有效内存。

    • 创建数组的时候，如果在创建的同时不初始化数组则必须指定大小

      new int[3];

    • 创建数组的时候如果不指定大小则必须在创建的同时初始化数组。

      new int[]{0, 1, 2}

  2. String类型的对象不可变，在每次对该对象进行修改的时候相当于新建一个新的String对象，然后再把指针指向新的对象。所以如果需要经常改变字符串，不要用String, 用StringBuilder。

StringBuilder sb 添加 1B 要先append(1)再append(B)

  1. ArrayList类, List类

ArrayList允许插入不同类型数据（代码第一行没有说明要插入什么类型数据）。ArrayList会把所有插入其中的数据都当作为object类型来处理。这样，在我们使用ArrayList中的数据来处理问题的时候，很可能会报类型不匹配的错误，也就是说ArrayList不是类型安全的。既使我们保证在插入数据的时候都很小心，都有插入了同一类型的数据，但在使用的时候，我们也需要将它们转化为对应的原类型来处理。这就存在了装箱与拆箱的操作，会带来很大的性能损耗。

为什么一般都使用 List list = new ArrayList() ,而不用 ArrayList alist = new ArrayList()呢？

问题就在于List有多个实现类，现在你用的是ArrayList，也许哪一天你需要换成其它的实现类，如 LinkedList或者Vector等等，这时你只要改变这一行就行了：List list = new LinkedList(); 其它使用了list地方的代码根本不需要改动。假设你开始用 ArrayList alist = new ArrayList(), 这下你有的改了，特别是如果你使用了 ArrayList特有的方法和属性 Ref:
