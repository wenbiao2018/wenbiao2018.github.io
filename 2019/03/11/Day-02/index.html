StringBuilder? StringBuffer? String?

String 类型的对象不可变，在每次对该对象进行修改的时候相当于新建一个新的String对象，然后再把指针指向新的对象。所以如果需要经常改变字符串，不要用String。StringBuilder是非线程安全的。

  • String 字符串常量
  • StringBuffer 字符串变量（线程安全）
  • StringBuilder 字符串变量（非线程安全)

Ref: String、StringBuilder、StringBuffer 用法比较

线程安全是什么

允许被多个线程同时执行的代码称作线程安全的代码, 多个线程同时获取同一个资源。

Ref: Java线程安全和非线程安全

为什么string.length()后面有括号，而int[]调用length不需要括号

需要加(）的是方法（或者叫函数），不加() 直接用名字就可以得到结果的叫属性；方法是对未知的东西去执行求结果，属性是对已知的东西读取出来。

java作用域public, private, protected 及不写时的区别

在说明这四个关键字之前，我想就class之间的关系做一个简单的定义，对于继承自己的class，base class可以认为他们都是自己的子女，而对于和自己一个目录下的classes，认为都是自己的朋友。

  1. public：public表明该数据成员、成员函数是对所有用户开放的，所有用户都可以直接进行调用
  2. private：private表示私有，私有的意思就是除了class自己之外，任何人都不可以直接使用，私有财产神圣不可侵犯嘛，即便是子女，朋友，都不可以使用。
  3. protected：protected对于子女、朋友来说，就是public的，可以自由使用，没有任何限制，而对于其他的外部class，protected就变成private。
