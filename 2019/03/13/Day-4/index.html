Java substring() 方法

substring(int beginIndex) or substring(int beginIndex, int endIndex) 方法返回字符串的子字符串。

beginIndex -- 起始索引（包括）, 索引从 0 开始。

endIndex -- 结束索引（不包括）。

Ref: Java substring() 方法

IndexOutOfBoundsException - if the beginIndex is negative, or endIndex is larger than the length of this String object, or beginIndex is larger than endIndex.

substring(length) will return an empty string Ref: substring(length)
