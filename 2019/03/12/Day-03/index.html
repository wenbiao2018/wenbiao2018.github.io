int溢出

代码： int a = 2147483647; System.out.println(a); int b = a + 1; System.out.println(b);

结果：

2147483647
-2147483648

Reason: int类型二进制存储的第一位为符号位，0表示正数，1表示负数，2147483647这个数字的的二进制表达为01111111111111111111111111111111,加1以后的值为10000000000000000000000000000000,而10000000000000000000000000000000表示的是-2147483648这个数字.

Ref:Java整形溢出处理机制
